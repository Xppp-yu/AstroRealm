## 知识要点

### Git 架构
- 四层结构：工作区 → 暂存区 → 本地仓库 → 远程仓库
- commit 是版本快照，branch 是独立开发线
- 安全恢复：`git checkout <commit> -- <目录>/`

### 混合 Git 策略
- 大仓库统一管理 + 单项目可独立推送
- 用分支隔离实验性修改

### Obsidian 双链
- `[[笔记名]]` 创建链接
- 形成知识网络
- MOC 作为知识图谱入口

### Claude Code Hooks 系统
- **SessionStart**: 对话开始时自动加载知识库状态
- **UserPromptSubmit**: 用户发消息时自动检索相关知识
- 脚本位置: `E:\CoBrain\scripts\`

### Memory vs CoBrain 分工
| 系统 | 作用 | 内容风格 |
|------|------|----------|
| Memory | 快速索引 | 简短、关键、机器优化 |
| CoBrain | 知识沉淀 | 详细、双链、人类友好 |
