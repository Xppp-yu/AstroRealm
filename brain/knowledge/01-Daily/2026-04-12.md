## 技术笔记

### Dataview 配置
需启用以下选项才能使用 DataviewJS：
```json
{
  "enableDataviewJs": true,
  "enableInlineDataviewJs": true
}
```

### CSS Snippet 启用
1. 设置 → 外观 → CSS 代码片段
2. 刷新并启用 `cobrain-home`
