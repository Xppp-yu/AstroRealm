# Obsidian

> 本地优先的知识管理工具，支持双向链接和知识图谱

## 概述
Obsidian 是一款基于本地 Markdown 文件的知识管理工具，核心特性是**双向链接**（Bi-directional Links），可以像维基百科一样构建知识网络。

## 核心特性

### 双向链接
使用 `[[笔记名]]` 语法创建链接：
- 自动建立双向引用关系
- 查看"反向链接"面板了解哪些笔记引用了当前笔记
- 形成知识网络而非孤立的文档

### 知识图谱
可视化展示笔记之间的链接关系，发现知识的隐藏关联。

### 本地优先
- 所有数据存储在本地文件夹
- 纯 Markdown 格式，不依赖特定软件
- 可用 Git 版本控制

## 在 CoBrain 中的应用
CoBrain 使用 Obsidian 作为知识库前端：
- 位置: `E:\CoBrain`
- 结构: 维基百科风格的三层知识体系
- 双链: 连接原子笔记、日志、沉淀知识

## 核心插件推荐
- **Graph View**: 知识图谱可视化
- **Backlinks**: 反向链接面板
- **Templates**: 模板快速插入
- **Dataview**: 数据查询和聚合
