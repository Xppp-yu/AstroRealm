# CoBrain 使用说明

## 这是什么？

CoBrain 是一个人机共脑知识库，利用 Obsidian 的双链特性连接知识点。

## 核心概念

### 双链语法
```markdown
[[笔记名称]]           # 链接到另一篇笔记
[[笔记名称|显示文本]]   # 自定义显示文本
#标签                  # 标签
```

### 笔记类型

| 类型 | 目录 | 模板 |
|------|------|------|
| 项目笔记 | 02-Projects/ | 项目笔记.md |
| 概念笔记 | 03-Concepts/ | 概念笔记.md |
| 决策记录 | 05-Decisions/ | 决策记录.md |
| 开发日志 | 01-Daily/ | 开发日志.md |
| 知识问答 | 01-Daily/ | 知识问答日志.md |

### 日志分类规则

**开发/项目类**：
- 简略记录完成内容
- 不需要代码细节
- 重点：目标、进度、问题

**知识问答类**：
- 详细记录知识点
- 建立概念链接
- 作为长期知识管理

## 与 Claude 的协作

1. **你在 Obsidian 写笔记** → 我能读取并理解
2. **我们对话** → 我自动生成/更新日志
3. **知识沉淀** → 双链连接形成知识网络
